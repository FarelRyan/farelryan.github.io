# Game Assets

Place your game images and rank icons here:

## Game Background Images (used as full card backgrounds)
Recommended: 600x900px or similar portrait ratio, high quality

- `cs2.jpg` — Counter-Strike 2 artwork/screenshot
- `valorant.jpg` — Valorant artwork/screenshot  
- `dota2.jpg` — DOTA 2 artwork/screenshot

## Rank Icons (shown in the rank badge area)
Recommended: 128x128px, PNG with transparency

- `cs2-rank.png` — Your CS2 rank icon
- `valorant-rank.png` — Your Valorant rank icon
- `dota2-rank.png` — Your DOTA 2 rank icon

## How to update ranks
Edit `src/components/GamingModal.jsx` and change the `rank` field for each game.
